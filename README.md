# South African Short-Term Insurance Underwriting & Risk Analytics

## 📊 Executive Project Overview
This project simulates an end-to-end data analytics workflow for a short-term personal lines insurance provider operating in South Africa. The objective is to identify regional underwriting margin leaks, calculate core portfolio loss ratios, and isolate fraudulent claim patterns for Special Investigative Unit (SIU) referral.

### 💡 Key Metrics & Insights
* **Total Gross Written Premium (GWP):** R142.5M
* **Overall Portfolio Loss Ratio:** 64.07% (Healthy market benchmark)
* **Underwriting Vulnerability:** Gauteng (GP) flagged with a critical **76.4% Loss Ratio**, requiring immediate premium recalibration due to high vehicle theft risk.
* **Fraud Detection:** Automated isolation filters successfully flagged high-value claims submitted within 14 days of policy inception.
